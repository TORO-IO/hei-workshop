What is a Preprocessor?

A preprocessor is a program that takes one type of data and converts it to another type of data.

- http://learn.shayhowe.com/advanced-html-css/preprocessors/

What is a CSS Preprocessor?

A scripting language that extends CSS and gets compiled into regular CSS syntax. Adding features that allow variables, mixins, functions and many other techniques that allow you to make CSS that is more maintainable, themable and extendable.

- https://drupalize.me/videos/what-css-preprocessor?p=1175

Why use one?

  • Cleaner code with reusable pieces
  • More flexibility to do things on the fly
  • Sharelable snippets and libraries
  • Easily produce CSS that works across browsers


- https://drupalize.me/videos/what-css-preprocessor?p=1175

In this demo we will use SCSS.

Sass 3 introduces a new syntax known as SCSS which is fully compatible with the syntax of CSS, while still supporting the full power of Sass.

Syntax

Variables


/* Sass (SCSS) */
$font-size: 16px;

div {
  font-size: $font-size;
}

/* LESS */
@font-size: 16px;

div {
  font-size: @font-size;
}

/* Stylus */
font-size = 16px

div
  font-size font-size

/* Compiled CSS */
div {
  font-size: 16px;
}

Nesting


/* Sass (SCSS) */
$link-color: #999;
$link-hover: #229ed3;

ul {
  margin: 0;

  li {
    float: left;
  }

  a {
    color: $link-color;

    &:hover {
      color: $link-hover;
    }
  }
}

Nesting


/* LESS */

@link-hover: #229ed3;

  margin: 0;

  li {
    float: left;
  }

  a {
    color: @link-color;

    &:hover {
      color: @link-hover;
    }
  }
}

Nesting


/* Stylus */
link-color = #999
link-hover = #229ed3

ul
  margin 0
  li
    float left

To apply KISS and DRY

We will use a CSS Preprocessor (Sass) 

/* using a mixin */
@mixin title($font-size: 48px) {
  padding: 10px;
  color: #333;
  font-size: $font-size;
  font-weight: 300;
  font-family: Open Sans,sans-serif;
  line-height: 40px;
  text-shadow: 0 0 0 #000;
}
.title-1 { @include title(); }
.title-2 { @include title(36px); }
.title-3 { @include title(28px); }








/* using a base class / extends */
.title {
  padding: 10px;
  color: #333;
  font-weight: 300;
  font-family: Open Sans,sans-serif;
  line-height: 40px;
  text-shadow: 0 0 0 #000;

  &.title-1 { font-size: 48px; }
  &.title-2 { font-size: 36px; }
  &.title-3 { font-size: 28px; }
}

  5. Create a file name it "Gulpfile.js"
  6. If app is not running, type "node server.js"

Place this code in the Gulpfile.js 


'use strict';

var gulp = require('gulp');
var sass = require('gulp-sass');

gulp.task('sass', function () {
  return gulp.src('./sass/**/*.scss')
    .pipe(sass().on('error', sass.logError))
    .pipe(gulp.dest('./public/css'));
});

gulp.task('sass:watch', function () {
  gulp.watch('./sass/**/*.scss', ['sass']);
});

function PostCtrl ($scope, $location, api) {
  $scope.blog = {
    title: '',
    content: ''
  }

  // replace the textarea to make it a real editor

  $scope.post = function () {

      .then( function(response) {
        $location.path( '/' );

        alert('Something went wrong');
      });
  }
}
